Questions fréquentes

Comment créer mon compte enseignant ?

Rendez-vous sur la page d'inscription et remplissez le formulaire. Un code de vérification sera envoyé à votre adresse email.

Comment ajouter mes élèves ?

Dans la gestion de classe, cliquez sur « Ajouter un élève » pour les ajouter manuellement. Vous pouvez aussi distribuer un code d'accès classe pour que vos élèves s'inscrivent eux-mêmes depuis l'application Élèves.

Comment les parents peuvent-ils suivre leur enfant ?

Les parents s'inscrivent sur l'application ProfCalendar Parents et rejoignent la classe avec le code fourni par l'enseignant. Ils peuvent consulter les notes, absences et justifier les absences.

Comment utiliser l'Apple Pencil pour annoter les PDF ?

Sur l'application iPad Enseignant, ouvrez un document PDF. Le support natif Apple Pencil vous permet d'annoter directement avec votre stylet. Seul le stylet dessine, votre doigt continue de naviguer normalement.

Comment fonctionnent les exercices et la gamification ?

Depuis l'Atelier, créez des exercices (QCM, réponses courtes) et publiez-les à vos classes. Le système RPG intégré permet de motiver vos élèves avec des points d'expérience, des niveaux et des combats de groupe en classe.

Comment fonctionne l'abonnement ?

ProfCalendar propose un accès gratuit avec les fonctionnalités de base. L'abonnement Premium débloque l'ensemble des outils avancés. Consultez notre page tarifs pour plus de détails.

Mes données sont-elles sécurisées ?

Oui. Vos données sont hébergées en Europe, chiffrées en transit et au repos. L'authentification à deux facteurs (2FA) est disponible. Consultez notre politique de confidentialité pour plus de détails.

Comment supprimer mon compte ?

Rendez-vous dans Paramètres > Supprimer mon compte. Toutes vos données seront définitivement supprimées.
